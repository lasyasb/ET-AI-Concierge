# 🤖 ET AI Concierge — Intelligent Financial Life Navigator

ET AI Concierge is a modern AI-powered fintech web application designed to help users discover, understand, and utilize financial services through a personalized conversational experience.

The platform acts as a smart financial guide that profiles users in minutes and recommends relevant tools, learning modules, credit services, and protection strategies.

---

## 🚀 Problem Statement

The Economic Times ecosystem offers a wide range of services — market insights, wealth tools, financial education, loans, insurance partnerships, and more.

However, most users explore only a small fraction of these offerings.

ET AI Concierge solves this by:

• Understanding user financial profile through a conversational onboarding
• Creating a personalized financial dashboard
• Recommending relevant ET ecosystem services
• Providing actionable financial tools and insights

## ⚙️ Installation & Setup

```bash
git clone https://github.com/YOUR_USERNAME/et-ai-concierge.git
cd et-ai-concierge
npm install
npm run dev
```

Create `.env` file and add Firebase credentials.
